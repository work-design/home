## 项目说明

我曾于国内一中医院校针灸专业就读，本专业课程包含中医、针灸、西医三个体系。在临床实习阶段，发现自身性格不容易hold住这个与患者斗智斗勇的事业，鉴于骨子里的技术宅气息，遂转行ruby程序员。

很不幸，上面所述的各个程序员常见疾病，我都得过，或轻或重。也经过自己的实践，取得了较好的康复效果。

推荐的商品为本人经过实践使用的产品，发现更具性价比的良心产品后我会增加推荐。
